Responda las siguientes preguntas del cuestionario 2

Flexbox

1. ¿Cuál es el propósito principal de Flexbox en CSS?

a. Crear animaciones y transiciones suaves.
b. Facilitar la creación de diseños responsivos.
c. Aplicar efectos de sombra y bordes a elementos.
d. Estilizar fuentes y tipografías.

2. ¿Qué propiedad de CSS se utiliza para convertir un contenedor en un contenedor flexible (flex container) en Flexbox?

a. display: block;
b. display: flex;
c. flex-mode: container;
d. flex-container: true;

3. ¿Cuál de las siguientes propiedades de CSS se utiliza para distribuir el espacio sobrante entre los elementos flexibles en Flexbox?

a. flex-align
b. flex-space
c. justify-content
d. flex-fill

4. ¿Qué propiedad de CSS se utiliza para controlar el tamaño de los elementos flexibles en Flexbox?

a. flex-basis
b. flex-size
c. item-size
d. flex-shrink

5. ¿Cuál de las siguientes afirmaciones es cierta sobre la propiedad "align-items" en Flexbox?

a. Se utiliza para controlar el orden de los elementos flexibles dentro del contenedor.
b. Alinea los elementos flexibles en el eje principal de la cuadrícula.
c. Distribuye el espacio sobrante entre los elementos flexibles.
d. Establece el tamaño predeterminado de los elementos flexibles.
